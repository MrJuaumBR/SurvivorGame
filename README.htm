About

  1. What is this?

    Is a simple game made with Python and PyGame, Using JPyDB.

  2. I Can add Plugins?

    Yes, for the first time in my life, i created a game with support to Plugins/Mods.

  3. I Can create Plugins?

    Yes, But you will need to follow all things into LICENSE.

  4. I Can distribuite the game?

    Maybe, if we(MrJuaum or ChosenBit) give permission to do it, you can read LICENSE to more informartion.

  5. The Plugins are free?

    Yes, all plugins need to be free.

  6. Can plugins be harmful?

    Perhaps, it depends on where you download, who creates it and their intentions, please check if he needs to download any data (usually only downloads his icon), if you are in doubt consult our Discord server
